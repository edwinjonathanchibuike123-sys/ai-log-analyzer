# AI Log Analyzer (Mini AIOps Project)

## Description
A simple Python-based log analysis tool that scans system logs, detects errors and warnings, and generates a summary report.

## Features
- Parses log files
- Detects errors, warnings, and info logs
- Generates summary insights

## How to Run
python3 app.py

## Use Case
Useful in DevOps environments to quickly identify system issues from logs.
